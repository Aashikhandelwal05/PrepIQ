# Recommended GitHub Labels

| Label | Color | Purpose |
| --- | --- | --- |
| `good first issue` | `#7057FF` | Small, well-scoped tasks for first-time contributors |
| `help wanted` | `#008672` | Tasks maintainers actively want community help with |
| `bug` | `#D73A4A` | Broken behavior or regressions |
| `enhancement` | `#A2EEEF` | Product or developer-experience improvements |
| `documentation` | `#0075CA` | README, docs, setup, and diagrams |
| `frontend` | `#1D76DB` | React, UI, UX, client-side state |
| `backend` | `#5319E7` | API, auth, data, service logic |
| `testing` | `#FBCA04` | Test coverage and automation |
| `devops` | `#0E8A16` | CI, Docker, deployment, workflows |
| `security` | `#B60205` | Secrets, auth, validation, abuse prevention |
| `accessibility` | `#C2E0C6` | Keyboard navigation, contrast, semantics |
| `design` | `#F9D0C4` | Visual polish and product design work |
| `priority: high` | `#B60205` | Important and urgent |
| `priority: medium` | `#FBCA04` | Important but not urgent |
| `priority: low` | `#0E8A16` | Nice-to-have improvements |
| `difficulty: easy` | `#C5DEF5` | Suitable for newcomers |
| `difficulty: medium` | `#FCE94F` | Requires some project context |
| `difficulty: hard` | `#D4C5F9` | Cross-cutting or architecture-heavy work |
| `mentor-needed` | `#E99695` | A maintainer should guide the contributor closely |
